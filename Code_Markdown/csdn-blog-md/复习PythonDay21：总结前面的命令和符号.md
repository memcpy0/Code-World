> 本节任务

2. 在每一个关键词和字符后面写出它的名字，并且说明它的作用。如果在书里找不到符号的名字，就上网找一下。如果你不知道某个关键字或者符号的作用，就回到用到该字符的章节通读一下，并且在脚本中测试一下这个字符的用处；
3. 找不到答案的东西，只要把这些记在列表里，它可以提示你还有哪些东西不懂；

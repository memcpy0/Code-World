随机化算法的一个用途在于——以 $\text{O(logN)}$ 期望时间支持查找和插入的数据结构，它对于任意输入序列的每次操作，运行时间的期望值都为 $\text{O(logN)}$ ，能够完成插入、删除和所有涉及排序的操作。

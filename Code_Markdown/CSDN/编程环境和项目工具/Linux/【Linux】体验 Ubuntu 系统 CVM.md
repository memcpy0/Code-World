## 使用 WebIDE 体验 CVM
使用文件浏览器
在 WebIDE 的左侧是 `文件浏览器` ，您可以通过 `文件浏览器` 操作 CVM 上的文件。
使用文档编辑器
在 WebIDE 中间的上部分是 `文档编辑器` ，您可以通过 `文档编辑器` 编辑 `文件浏览器` 选中的 CVM 上的文件。
使用 shell 终端
在 WebIDE 中间的下部分是 `shell` 终端，您可以通过 `shell` 终端在 CVM 上执行各种 `shell` 命令。

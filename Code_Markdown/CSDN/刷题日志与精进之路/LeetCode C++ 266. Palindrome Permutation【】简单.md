
Given a string, determine if a permutation of the string could form a palindrome.

**Example 1:**

```haskell
Input: "code"
Output: false
```

**Example 2:**

```haskell
Input: "aab"
Output: true
```

**Example 3:**

```haskell
Input: "carerac"
Output: true 
```

题意：给定一个字符串，判断该字符串中是否可以通过重新排列组合，形成一个回文字符串。
